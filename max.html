FIGHTER - RUSHER TYPE
Surge - use to attack twice early on
Projectile Deflect - when being healed this makes it harder to be picked off
Disarm - can be used with surge
Circle Sweep - good splash
In battle: Run at enemies, use surge so you can use disarm and circle sweep (if disarmed enemy has harder time attacking), attack until low health, drop back and be healed
